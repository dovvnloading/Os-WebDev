View Repository Explore Features

Democratizing Creation

This project acts as a functional prototype intended to break down the barriers of entry for web development and design. By integrating advanced artificial intelligence directly into the workspace, Os-WebDev aims to allow individuals to build complex logic and interfaces regardless of their prior coding experience.

See it in Action

Agentic Workflow

Orchestrates a team of AI agents. The Architect (Gemini 2.5) plans the strategy, and the Developer (Gemini 3 Pro) executes high-fidelity code.

Virtual File System

A fully in-memory client-side file system. Create, delete, and edit HTML/CSS/JS files with dynamic dependency injection into the preview.

Shadow Protocol

Uses a hidden XML protocol <file> to inject code directly into the IDE, keeping your chat interface clean and conversational.

Adaptive Aesthetics

Seamless transitions between light and dark claymorphic themes.

Get Started Locally

bash

# 1. Clone the repository

git clone https://github.com/dovvnloading/Os-WebDev.git
cd Os-WebDev

# 2. Install dependencies

npm install

# 3. Setup Environment (.env)

GEMINI_API_KEY=your_api_key_here

# 4. Run Development Server

npm run dev
